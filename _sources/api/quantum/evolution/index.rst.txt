.. py:function:: time_evolution(hamiltonian: orquestra.quantum.wip.operators.PauliRepresentation, time: Union[float, sympy.Expr], method: str = 'Trotter', trotter_order: int = 1) -> orquestra.quantum.circuits.Circuit

   Create a circuit simulating evolution under given Hamiltonian.

   :param hamiltonian: The Hamiltonian to be evolved under.
   :param time: Time duration of the evolution.
   :param method: Time evolution method. Currently the only option is 'Trotter'.
   :param trotter_order: order of Trotter evolution (1 by default).

   :returns: Circuit approximating evolution under `hamiltonian`.
             Circuit's unitary i approximately equal to exp(-i * time * hamiltonian).


.. py:function:: time_evolution_for_term(term: orquestra.quantum.wip.operators.PauliTerm, time: Union[float, sympy.Expr]) -> orquestra.quantum.circuits.Circuit

   Evolves a Pauli term for a given time and returns a circuit representing it.
   Based on section 4 from https://arxiv.org/abs/1001.3855 .
   :param term: Pauli term to be evolved
   :param time: time of evolution

   :returns: Circuit representing evolved term.
   :rtype: Circuit


.. py:function:: time_evolution_derivatives(hamiltonian: orquestra.quantum.wip.operators.PauliRepresentation, time: float, method: str = 'Trotter', trotter_order: int = 1) -> Tuple[List[orquestra.quantum.circuits.Circuit], List[float]]

   Generates derivative circuits for the time evolution operator defined in
   function time_evolution

   :param hamiltonian: The Hamiltonian to be evolved under. It should contain numeric
                       coefficients, symbolic expressions aren't supported.
   :param time: time duration of the evolution.
   :param method: time evolution method. Currently the only option is 'Trotter'.
   :param trotter_order: order of Trotter evolution

   :returns: A Circuit simulating time evolution.
