orquestra.quantum.utils.scale_and_discretize(values: Iterable[float], total: int) → List[int]#

Convert a list of floats to a list of integers such that the total equals a given value and the ratios of elements are approximately preserved.

Parameters

  • values The list of floats to be scaled and discretized.

  • total The desired total which the resulting values should sum to.

Returns

A list of integers whose sum is equal to the given total, where the

ratios of the list elements are approximately equal to the ratios of the input list elements.
