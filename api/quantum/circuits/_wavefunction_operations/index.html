class quantum.circuits._wavefunction_operations.MultiPhaseOperation#

Operation applying distinct phase to each wavefunction component.

MultiPhaseOperation with parameters theta_1, theta_2, …. theta_2^N, transforms a N qubit wavefunction (psi_1, psi_2, …, psi_2^N) into (exp(i theta_1)psi_1, exp(i theta_2) psi_2, …, exp(i theta_2^N) psi_2^N).
